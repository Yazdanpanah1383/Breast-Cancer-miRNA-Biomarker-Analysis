# Breast-Cancer-miRNA-Biomarker-Analysis
Bioinformatics analysis of breast cancer serum miRNA biomarkers using R and limma.

# 🧬 Breast Cancer miRNA Analysis

## 📌 Project Overview
This project analyzes serum microRNA (miRNA) expression profiles in breast cancer using the GEO dataset GSE73002.

The goal is to identify differentially expressed miRNAs and evaluate their potential as diagnostic biomarkers.

---

## 📊 Dataset Information
- Source: NCBI GEO (GSE73002)
- Sample Type: Serum
- Groups:
  - Breast Cancer
  - Benign Breast Disease
  - Non-cancer Controls

---

## 🧪 Workflow
1. Data Preprocessing
2. Differential Expression Analysis (limma)
3. Feature Selection
4. Visualization (Volcano, Heatmap, PCA, Boxplot)
5. ROC Analysis

---

## 📌 Key Findings
- miR-17-5p showed elevated expression in cancer samples
- let-7a-5p showed decreased expression patterns
- AUC > 0.9 using combined miRNA signatures
